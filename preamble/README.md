# Преамбула для оформления отчетов/РПЗ по ГОСТу в МГТУ им. Н. Э. Баумана.

## Краткое описание

Попытка собрать преамбулу из уже имеющихся в различных репозиториях и адаптировать ее под последние (25-26 год) требования кафедры.
Именно с этой преамбулой я планирую писать РПЗ к курсовой работе по КГ, а также ВКР, если доживу. \
**Внимание:** в любой момент времени преамбула может перестать соответствовать требованиям кафедры, если они вдруг изменятся.
Данную преамбулу вы используете на свой страх и риск, **у кафедры нет официальных шаблонов**.

### Личный опыт использования
- На момент окончания пятого семестра представленная в данном репозитории преамбула активно использовалась мною 
    по ходу сема во всех письменных работах, а именно: лабах, курсаче, нире. Все работы сданы и защищены без
    проблем, нормоконтроль (у Мальцевой Д. Ю.) пройден дважды, для курсача и для нира.
- Могут возникать проблемы при использовании данной преамбулы с техстудией, по крайней мере оные наблюдались у некоторых
    одногруппников. Сам техстудию не использую ибо ультра кринж.

## Как использовать?

В примере рассматривается следующая структура проекта, при желании можно отойти от нее, но необходимо будет поменять
пути в паре мест:
```
.
└── report
    ├── auto
    │   ├── gost705.bst
    │   └── preamble.tex
    ├── img
    │   └── example.png
    ├── report.pdf
    ├── report.tex
    ├── makefile
    └── parts
        ├── 00-summary.tex
        ├── 01-definitions.tex
        ├── 02-introduction.tex
        ├── 03-analytical.tex
        ├── 04-design.tex
        ├── 05-classification.tex
        ├── 06-conclusion.tex
        ├── 07-bibliography.tex
        └── bibliography.bib
```

1. В первую очередь необходимо скачать репозиторий целиком или конкретно преамбулу. Думаю тут и без подсказок осилите :\).
2. Затем необходимо подключить преамбулу в начале своего документа.
    report.tex:
    ```latex
    \documentclass{...}
    
    \input{auto/preamble}

    \begindocument{document}
    ...
    ```
