# Компиляция LaTeX отчетов в PDF

## Краткое описание
Для написания всех документов в Латехе было решено использовать в первую очередь адекватный текстовый редактор, а не
техстудию. В связи с этим пришлось устанавливать все компоненты (аж целый `texlive-full`) отдельно и понадобилось организовать
какую-то +- адекватную систему сборки, чтобы не писать каждый раз все команды ручками. В качестве системы сборки был выбран `make`, 
так как он крайне прост и легок, да и подходит для Латеха очень даже неплохо.

Данный `makefile` поддерживает сборку с библиографией через `bibtex`, также в нем включена функция сжатия и вычистки мусора из итогового
документа с помощью `gs`. Данное поведение требовал сэр Костридзе для отчетов по АА, на моей памяти пару раз ломались шрифты на сервере с АА,
но в целом все нормально. При желании можно отключить, поменяв цель на `all`.

## Как использовать?

Мейкфайл работает с проектами следующей структуры:
```
.
└── report
    ├── auto
    │   ├── gost705.bst
    │   └── preamble.tex
    ├── img
    │   └── example.png
    ├── report.pdf
    ├── report.tex
    ├── makefile
    └── parts
        ├── 00-summary.tex
        ├── 01-definitions.tex
        ├── 02-introduction.tex
        ├── 03-analytical.tex
        ├── 04-design.tex
        ├── 05-classification.tex
        ├── 06-conclusion.tex
        ├── 07-bibliography.tex
        └── bibliography.bib
```

1. В первую очередь необходимо скачать репозиторий целиком или конкретно преамбулу. Думаю тут и без подсказок осилите :\).
2. Затем помещаем `makefile` в необходимое место и выполняем сборку:
    ```bash
    $ make
    ```
